# "To-do List"

## Description
A simple task management web application built with Flask. Users can add, edit, and delete tasks. This is my first website developed using the Flask framework.

## Objectives
- Learn the fundamentals of Flask and its core features.
- Understand how routing, templates, and databases work in a web application.
- Gain experience in developing a CRUD (Create, Read, Update, Delete) application.
- Strengthen my Python and web development skills.
